Swift-类和结构体

发表于 2017-09-28 | 分类于 iOS

Swift中结构体和类的相同点

  • 均可以定义属性和方法
  • 使用构造函数来初始化成员变量
  • 使用扩展来增加自身功能
  • 遵守协议

Swift中结构体和类的不同点

  • 结构体是值类型，赋值时是值拷贝
  • 类是引用类型，赋值只是指针拷贝，但指向的还是同一块内存地址
  • 类能够通过继承扩展自身功能，结构体不可以

  • 结构体有默认的初始化每一个成员变量的初始化函数，类没有

HTTP

Sep 1, 2014
开发工具 Xcode

开发iphone或是MAC应用的IDE；AppStore更新或是下载经常会出现缓慢现象，如果有开发者账号可以直接在开发者网站直接下载https://developer.apple.com/download/；通常速度会快许多；

开发语言 Objective-C

Objective-C是一种通用、高级、面向对象的编程语言。它扩展了标准的ANSI C编程语言，将Smalltalk式的消息传递机制加入到ANSI C中。目前主要支持的编译器有GCC和Clang（采用LLVM作为前端）。
Objective-C的商标权属于苹果公司，苹果公司也是这个编程语言的主要开发者。苹果在开发NeXTSTEP操作系统时使用了Objective-C，之后被OS X和iOS继承下来。现在Objective-C是OS X和iOS操作系统、及与其相关的API、Cocoa和Cocoa Touch的主要编程语言。（wiki百科）

模型-视图-控制器
  • 视图对象：app界面上所能看到的控件（按钮，文本框，标签等），是构成用户界面的元素；
  • 模型对象：映射为现实中的事物，用来存储数据信息；
  • 控制器对象：充当视图对象与模型对象之间交流的中介；
MVC设计模式

用户触发屏幕上的一个按钮，作为视图模型按钮向控制器对象发送消息，此时控制器对象会根据消息的内容选择更新模型数据，当模型数据更新后他会主动通知控制器对象，得到通知消息的控制器对象会向视图对象发送更新视图的消息，此时界面会显示更新后的内容，这就是MVC一个消息传递的流程

程序包（bundle）

程序包其实就是目录，其中包含应用的可执行文件和程序运行所需要的所有资源文件

应用图标

在主屏幕中指代一个应用的图片，目前通用的图标大小为 120*120px；在项目中可以在Image.xcasets中找到AppIcon将设计的图标直接拖到指定尺寸的位置；

启动图片

启动图片的主要作用是向用户传达app正在启动中，目前iOS7以上存在 640*960 、 768*1136两种尺寸

真机安装app
  • App ID : 项目创建的时候会有一个bundle Id 这个表示类似于身份证号一样，它作为应用在AppStore中的唯一标识；
  • Developer Certificate:开发者证书，这个在开发者网站上创建（前提是已经购买开发者账号），Xcode会使用此证书为代码做签名；
  • Provisioning Profile: 开发者mac电脑和测试设备都有这个文件，这个文件包含三部分：一个开发者证书、一组测试设备的UDID、一个应用的App Id；这有这些条件都满足才能将app安装到测试设备中
  • ipa直接安装到手机：ipa包中的Provisioning Profile文件中包含一组UDID，和测试设备比对包含就能安装，否则安装失败；

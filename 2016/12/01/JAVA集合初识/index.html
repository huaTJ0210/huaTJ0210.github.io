JAVA集合初识

2016-12-01 | JAVA

基本数组

基本数组在创建的时候分配固定大小的存储空间，可以根据index来改变存储的value；它的适用的场景是已知存储对象的个数，业务中不在对其扩充存储空间的情况；

//基本数组的使用
    String[] array = new String[100];
    array[0] = "bike";
    array[1] = "car";
    int arrayLength = array.length;
    System.out.println(arrayLength);

ArrayList

当业务需求列表需要存储的对象会动态增加时，使用基本数组就不能满足业务的需求；JAVA中提供了ArrayList，它实质是一个管理数组的manager，提供了一些API可以对管理的数组进行增删改查，同时可以对数组的存储空间进行动态增加操作； 

ArrayList<String> lists = new ArrayList<>();
lists.add("apple");
lists.add("banana");
String apple = lists.get(0);
lists.set(1,"orange");
lists.remove(0);
int appleIndex=lists.indexOf("apple");
int size = lists.size();
lists.trimToSize();

LinkedList

使用数组存储元素的时候在碰到删除数组中间元素的需求时，会有很大的性能问题，API内部需要将删除位置以后的元素都要移动位置，存储的元素太多这种操作就显得没有必要；数据结构中为了弥补数组的这种缺点采用链表来存储数据，JAVA中提供了LinkedList来供日常的使用； 

// 在链表尾部添加元素，并根据迭代器获取值
List<String> staff = new LinkedList<>();
staff.add("Amy");
staff.add("Bob");
staff.add("Carl");
Iterator<String> iter = staff.iterator();
String first = (String) iter.next();
String second = (String) iter.next();
System.out.println(first+"--"+second);

// 使用ListIterator在链表的指定位置插入元素，
List<String> staffNext = new LinkedList<>();
staffNext.add("Amy");
staffNext.add("Bob");
staffNext.add("Carl");
ListIterator<String> iterNext = staffNext.listIterator();
iterNext.next();
iterNext.add("hz");//在第二个位置之前增加元素
StringBuilder names = new StringBuilder();
for (String string : staffNext) {
    names.append(string+"/");
}
System.out.println(names);

散列集

遇到不在意存储元素的顺序，快速查找元素的需求时可以使用 hashSet; 

Set<String> words = new HashSet<>();
words.add("a");
words.add("b");

树集

TreeSet和 hashSet相似只不过TreeSet会对加入的元素进行排序，遍历的时候以指定顺序输出

队列

队列可以让编程者有效的在队列头部删除一个元素，在队列的尾部新增一个元素； 

PriorityQueue<String> queue = new PriorityQueue<String>();
queue.add("first");
queue.add("second");
queue.add("third");
//如果队列没有满，将给定的元素添加到这个双端列表的尾部，如果满了则会抛出异常
queue.offer("frist");
// 如果队列不为空，删除并返回这个队列头部的元素，如果为空抛出异常
queue.poll();
// 如果队列不为空，返回这个队列头部的元素，如果为空抛出异常
queue.peek();

散列表

散列表支持key-value的形式存储数据元素 

//不考虑顺序性可以使用HashMap
   Map<String,String> staffs = new HashMap<>();
   staffs.put("1", "jim");
   staffs.put("2", "tom");
   staffs.put("3", "david");
   staffs.get("1");// 根据指定key值获取value
   staffs.remove("3");// 根据key值移除value
   // 考虑顺序存储使用 TreeMap
   Map<String,String> staffsTree = new TreeMap<>();
   staffsTree.put("1", "jim");
   staffsTree.put("2", "tom");
   staffsTree.put("3", "david");

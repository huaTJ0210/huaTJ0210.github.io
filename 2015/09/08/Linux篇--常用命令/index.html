Linux篇--常用命令

Sep 8, 2015

ls

  • ls 显示当前目录下所有文件名
  • ls -l显示当前目录下所有文件的详细（文件权限、文件所有者，文件修改日期，文件名等）
  • ls -a显示当前目录下所有文件名包含隐藏文件

    cd

  • cd dirName or dirPath 切换到指定目录
  • cd /切换到根目录
  • cd ..切换到上一级目录
  • cd ~切换到当前主目录
  • cd -返回进入此目录之前所在的目录

pwd

pwd 查看当前完整路径

mkdir

mkdir dirName 创建一个名称为dirName的文件夹

rm

  • rm filename删除指定文件
  • rm -r dirName删除指定目录

touch

touch fileName 创建一个不存在的文件

cat

cat fileName 查看一个文件

mv

  • mv file1 file2 将文档 file1，更改档名为 file2.
  • mv file1 dir1 将文档 file1，移到目录 dir1 下，档名仍为 file1.
  • mv dir1 dir2 若目录 dir2 不存在，则将目录 dir1，及其所有档

vim

  • vim hua.txt 没有hua.txt 则创建
  • I 键 进入编辑状态
  • esc键退出工作状态
  • ZZ 键保存并退出
  • wq保存并退出
  • vim 使用详解

ping

ping ip地址测试与目标主机的连通性
